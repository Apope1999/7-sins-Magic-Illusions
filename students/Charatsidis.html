In the creation of the website i used the following HTML and CSS commands

  • "h1,h6". In order to define header size.
  • "img". In order to display an image which you link with src
  • "li". Creates a line.
  • "ol". For an ordered list.
  • "ul". Unordered list.
  • "name". Set an individual name to an object.
  • "target=". Sets a target iframe or place where the link you apply will open.
  • "usemap=". The content you applied it to uses the map of that name
  • "shape=polygon". Defines a shape in a map tag.
  • "map". Defines a map on an image if you provide the coordinates where you can link content.
  • "iframe". Either for a video or a small box where you can store visual content with links.
  • "coords". Defines the map coordinates on an image.
  • "div". Shows something as a block.
  • "a href". Adds attribute href which is used to create links.
  • "alt=". In order to display a text if the photograph is missing.
  • "frameborder=". To set the frameborder pixels of a video.
  • "allowfullscreen". To allow a video to open up full screen size through my website.
  • CSS "opacity". To change opacity of an image
  • CSS "fonts". To change the fonts of a paragraph.
  • CSS "position". To define the position of content.
  • CSS "height,widht". To define height and width of content.

The following javascript was used to resize the iframes "function resizeIframe(obj) { obj.style.height = obj.contentWindow.document.body.scrollHeight + 'px'; }" .It works paired with onload="resizeIframe(this);" in the iframe you wish to resize.

The BOOTSTRAP specific commands used were the following :

  • class="rounded-circle img-fluid" . Which would make images look rounded like a circle
  • class="row align-items-center" and "text-center". The first one centers the contents compared to the container height and the second centers compared to weight.
  • class="row justify-content-center". Puts the content in the middle of the container.
  • class="container-fluid" . It makes the contents resposnsive in different sizes of screens.
  • class="col-md-x" . Creates bootstrap columns that divide the width in order to separate content.
  • class="img-fluid". makes an image responsive to the container and changes its size appropriately for each screen size.
  • class="embed-responsive embed-responsive-4by3" . Makes the video with class="embed-responsive-item" responsive to screen size if its inside it.
  • class="navbar-". Defines various elements of a navigation bar.
