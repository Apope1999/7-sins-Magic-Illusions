In the creation of the website I used the following HTML and CSS commands:

HTML:

  • "h1","h6". Defines header size and level.
  • "img scr". Displays an image.
  • "ol". Creates an ordered list.
  • "li". Creates a line.
  • "id". Set an id to an object.
  • "target". Sets a target iframe or place where the link you apply will open.
  • "usemap". The content you applied it to uses the map of that name.
  • "map". Defines a map on an image if you provide the coordinates where you can link content.
  • "iframe". Either for a video or a small box where you can store visual content with links.
  • "div". Shows something as a block.
  • "a href". Adds attribute href which is used to create links.
  • "alt". Displays a text if the photograph is missing.
  • "style". Applies styles in-line.
  • "class". Sets a class to the group of objects.
  • "span". Provides a way to add a hook to a part of a text or a part of a document.
  • "figure". Specifies content such as images.

CSS:

  • "opacity". Change opacity of an image
  • "fonts". Changes the fonts of a paragraph.
  • "position". Defines the position of content.
  • "height,widht". Defines height and width of content.
  • "padding". Specifies the padding of an object.
  • "text-align". Aligns text to the given parameters e.g: center.
  • "float". Specifies how an object should float e.g: left.
  • "margin". Specifies the margin of an object.
  • "display". Specifies how a displayed object behaves.
  • "overflow". Specifies an action to be done with an overflowing part e.g: hidden.
  • "transform". Helps to rotate, scale, move, etc. the object.
  • "background-color". Specifies a background color of an object.
  • "font-size". Specifies a font size of a text.
  • "font-family". Specifies a font family for a text.

The BOOTSTRAP specific commands used were the following :

  • class=row. Creates bootstrap rows that divide height in order to separate content.
  • class=col-x-x. Creates bootstrap columns that divide the width in order to separate content.
  • class=d-none d-md-block. Hides a row/column above/below a certain resolution.
  • class=container. Contains large parts of the webpages.
