In the creation of the website i used the following HTML and CSS commands and JS code:

HTML:

  • “h1”,”h2": Defines header size and level.
  • "img scr": Displays an image.
  • "id": Set an id to an object.
  • "target": Sets a target iframe or place where the link you apply will open.
  • "iframe": Either for a video or a small box where you can store visual content with links.
  • "div": Shows something as a block.
  • "a href": Adds attribute href which is used to create links.
  • "alt”: Displays a text if the photograph is missing.
  • “style”: Applies styles in-line.
  • “class”: Sets a class to the group of objects.
  • "script": Define a client-side script (JavaScript). It either contains scripting statements, or it points to an external script file through the src attribute.

CSS:

  • "fonts": Changes the fonts of a paragraph.
  • "height,widht": Defines height and width of content.
  • “padding”: : Defines the ammount of padding assigned to an elementDefines the ammount of padding assigned to an element
  • “text-align”: Changes the allignement of text
  • “margin”: Defines the ammount of margin assigned to an element
  • “display” Defines whether the element will display or not
  • “background-color”: Changes the color of the element's baclground
  • “font-size”: defines the font size
  • “font-family”: defines the type of font used

JS

  • document.getElementById(id): js method to find an element in html

  • If...Else Statements: Conditional statements are used to perform different actions based on different conditions.

The BOOTSTRAP specific commands used were the following :

  • class=”row”. Creates bootstrap rows that divide height in order to separate content.
  • class="col-x-x" . Creates bootstrap columns that divide the width in order to separate content.
  • class="d-none d-md-block". Hides a row/column above/below a certain resolution.
  • class=”container”. Contains large parts of the webpages.
